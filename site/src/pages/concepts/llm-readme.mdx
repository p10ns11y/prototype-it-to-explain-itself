# LLM Prototypes Guide

Everything in this collection is built around one big idea:

> Look at the text so far. Predict the next token. Append it. Repeat.

The base `simple_llm_prototype.py` makes that loop completely visible with a ~150k parameter character-level LSTM.

From there we add layers **only through the Predictor seam** (`prompt → text`).

## The Sequence (as of now)

1. **Mini ReAct** — the control loop + the Predictor abstraction.
2. **Tool-Use Reliability Lab** — measurement of format and argument correctness.
3. **Memory** — STM + simple LTM with `extra_context` injection.
4. **Trajectory Evaluator** — outcome + process scoring + weak self-judge.
5. **Local Inference Playground** — proving the seam works with multiple backends.
6. **Synthetic Data Factory** — generate → critique → improve the model.
7. **Typed Agent Workflow** — make invalid states hard or impossible.
8. **Human-in-the-Loop Desktop** (terminal edition) — oversight patterns.
9. **Multi-Agent Debate** — the capstone (orchestrator + specialists + synthesis).
